    <article class="markdown-body entry-content p-3 p-md-6" itemprop="text"><h3><a id="user-content-udacity-deep-reinforcment-learning-nanodegree" class="anchor" aria-hidden="true" href="#udacity-deep-reinforcment-learning-nanodegree"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Udacity Deep Reinforcment Learning Nanodegree</h3>
<h1><a id="user-content-project-1-navigation" class="anchor" aria-hidden="true" href="#project-1-navigation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Project 1: Navigation</h1>
<h3><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Introduction</h3>
<p>For this project, you will train an agent to navigate (and collect bananas!) in a large, square world.</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://user-images.githubusercontent.com/10624937/42135619-d90f2f28-7d12-11e8-8823-82b970a54d7e.gif"><img src="https://user-images.githubusercontent.com/10624937/42135619-d90f2f28-7d12-11e8-8823-82b970a54d7e.gif" alt="Trained Agent" title="Trained Agent" style="max-width:100%;"></a></p>
<p>A reward of +1 is provided for collecting a yellow banana, and a reward of -1 is provided for collecting a blue banana.  Thus, the goal of your agent is to collect as many yellow bananas as possible while avoiding blue bananas.</p>
<p>The state space has 37 dimensions and contains the agent's velocity, along with ray-based perception of objects around agent's forward direction.  Given this information, the agent has to learn how to best select actions.  Four discrete actions are available, corresponding to:</p>
<ul>
<li><strong><code>0</code></strong> - move forward.</li>
<li><strong><code>1</code></strong> - move backward.</li>
<li><strong><code>2</code></strong> - turn left.</li>
<li><strong><code>3</code></strong> - turn right.</li>
</ul>
<p>The task is episodic, and in order to solve the environment, your agent must get an average score of +13 over 100 consecutive episodes.</p>
<h3><a id="user-content-files-and-folders" class="anchor" aria-hidden="true" href="#files-and-folders"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Files and Folders</h3>
<ul>
<li>navigation.py: the main routine for the basic-banana project. It is the high-level calls to construct the environment, agent, train and play.</li>
<li>visual_navigation.py: the main routine for the challenge project - visual banana.</li>
<li>dqn.py: the deep reinforcement learning algorithm routine.</li>
<li>dqn_agent.py: the class definitions of the agent and replay buffer.</li>
<li>model.py: the deep neural network models are defined in this file.</li>
<li>Environment.py: a wrapper for the UnityEnvironment. The wrapper makes the
environment interface similar to the OpenAI gym environment, so that
the DQN routines are more general.</li>
<li>dqn_test.py: a dqn test routine using the OpenAI gym environment (can be ignored).</li>
<li>folder <code>basic_banana</code>: the directory to save the unity environment run time of the basic agent in
sensor mode.</li>
<li>folder <code>visual_banana</code>: the directory to save the unity environment run time of the agent in
pixel/visual mode.</li>
<li>PlotScores.ipynb: the routine to plot the scores.</li>
<li>basic_banana_dqn_checkpoint.pth: saved weights for basic banana.</li>
<li>visual_banana_dqn_checkpoint.pth: saved weights for the challenge project. (achieved average score 13.03 in 985 episodes)</li>
</ul>
<h4><a id="user-content-environment-wrapper-class" class="anchor" aria-hidden="true" href="#environment-wrapper-class"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Environment Wrapper Class</h4>
<p>The <code>CollectBanana</code> Class is a wrapper for the <code>UnityEnvironment</code> class, which
includes the following main methods:</p>
<ul>
<li>step</li>
<li>reset</li>
<li>get_state</li>
<li>close</li>
</ul>
<p>The <code>name</code> parameter in the
constructor allows the selection of <code>state</code> format returned:</p>
<ul>
<li>
<p>For basic banana, the state is a 37-dimensional vector.
<code>self.state = self.env_info.vector_observations[0]</code></p>
</li>
<li>
<p>For visual banana, the state contains four frames by calling <code>get_state()</code>.
To fit the PyTorch format, the original frame format is transposed from NHWC (Batch, Height, Width, Channels) to NCHW
by numpy transpose function as follows:
<code>frame = np.transpose(self.env_info.visual_observations[0], (0,3,1,2))</code>
The current frame, together with previous frame <code>last_frame</code> and the second previous frame, <code>last2_frame</code>
are then assemblied into variable <code>CollectBanana.state</code> variable.</p>
</li>
</ul>

<p>From the challenge project, I observed the following:</p>
<ul>
<li>Color images are better than than the gray scale images, because the bananas are in yellow and blue, also
the background wall, floor, and sky. It is hard to distinquish them with only gray pictures.</li>
<li>Using multiple frames instead of single one. I used four frames and it improves the results a lot.</li>
<li>Using 3D convolutional layers yields bettter results than 2D in my test. That perhaps because the
input state are multiple images and temporal relationships. 3D filters are keen to detect the local
motion patterns.</li>
<li>The number of filters cannot be too small. I used 128, 256, 256 in these layers, which are much
better than small numbers that I tried earlier, such as 32, 64, 64.</li>
</ul>
<p>For future improvements - the challenge project:</p>
<ul>
<li>The agent could be able to achieve higher score if trained longer</li>
<li>Using other advanced DQN, such as double, dueling, replay buffer prioritization, rainbow, etc.</li>
</ul>
